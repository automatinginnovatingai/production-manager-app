About Windows SmartScreen

Our applications are completely safe and digitally clean. Because we distribute our software directly (not through the Microsoft Store), Windows SmartScreen may show a message the first time you install it.

This is a standard Windows precaution for newly released applications.

To continue with the installation:

- Select More info
- Choose Run anyway
After the first few installs, Windows automatically recognizes the application and the message disappears.


FAQ: Windows SmartScreen & Installation

Why do I see a Windows SmartScreen message when installing the app?
Windows shows SmartScreen messages for any new or independently distributed software. It’s a standard precaution, not an indication of a problem.

Is the application safe?
Yes. Our software is fully safe, clean, and used by professionals in the field. The message appears simply because the app is newly released and not yet recognized by Microsoft’s reputation system.

How do I proceed with the installation?
If SmartScreen appears:
1. Click “More info”
2. Select “Run anyway”
After a few installs across our user base, Windows automatically recognizes the app and the message disappears.

Why doesn’t this happen with big-name software?
Large vendors have long-standing reputation with Microsoft and distribute through the Microsoft Store or use extended validation certificates. New or custom business software often triggers SmartScreen until reputation builds.

Will this warning go away permanently?
Yes. As more clients install and use the software, Windows SmartScreen builds trust and stops showing the message.
